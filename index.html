Pengalaman saya dalam Pembelajaran Jarak Jauh (PJJ) selama Pandemi Covid-19 ini adalah saya merasa sangat berat dalam menjalaninya terutama pada saat awal - awal masa pandemi. Sebagai seorang pelajar saya harus jadi lebih mandiri lagi daripada sebelumnya, dimana saya harus rajin mencari informasi mengenai pembelajaran dan lebih memberanikan diri untuk bertanya agar tidak ada kesalahpahaman. Saya juga harus fokus pada saat ada pertemuan virtual, sebenarnya saya merasa itu berat sekali dan cukup membosankan. Tetapi itu sudah menjadi bagian tanggung jawab saya sebagai seorang pelajar. Saya juga mengalami kesulitan dalam hal teknologi, pada awalnya saya sempat merasa bingung sekali. Namun,lama kelamaan saya merasa lebih update lagi mengenai perkembangan teknologi. Saya banyak belajar dan bertanya kepada guru maupun teman. Selain itu, yang cukup menyulitkan adalah kendala jaringan dan listik. Hal ini cukup mengganggu dan dapat memecah konsentrasi jika terjadi pada saat ulangan maupun quiz dan sudah mendekati waktu pengumpulan.

Saya juga merasa bingung untuk pembelajaran praktik, dimana kita hanya dapat menonton video dan tidak dapat melakukannya secara langsung. Saya kesulitan dalam membayangkan bagaimana nanti saat praktik, karena jika hanya melihat video saja seperti terlihat mudah dan sebentar. Hal ini membuat saya sedikit takut dan tidak percaya diri, saya khawatir tidak dapat melakukan praktiknya dengan baik. Namun, sekarang saya dapat melihat sisi positifnya yaitu saya dapat belajar secara teori lebih dalam dan saya berharap ini dapat membantu saat nanti saya masuk dan melaksanakan praktik. Agar mengurangi kemungkinan kesalahan yang terjadi. Sisi positif dari belajar di rumah adalah saya dapat membantu ibu saya dalam pekerjaan rumah.
